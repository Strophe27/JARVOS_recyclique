# Matrice de parite UI — initialisation (pilotes Legacy -> Peintre_nano)

Date : 2026-04-10

## Objet

Amorcer la matrice normative de parite observable entre le frontend legacy (`recyclique-1.4.4`, reference UX sur `http://localhost:4445`) et `Peintre_nano`, avec trois lignes pilotes stables.

Les cellules non encore verifiees restent explicitement ouvertes ; aucune URL, `operationId`, cle `CREOS` ou preuve n'est inventee ici.

**Cadrage** : `references/artefacts/2026-04-10_02_cadrage-parite-ui-legacy-peintre.md`

## Regles d'usage

1. Le legacy sur `http://localhost:4445` sert a observer routes, shell, chrome et etats visibles ; toute precision manquante est renseignee apres verification sur cette instance, pas par supposition.
2. Toute ligne qui tend vers `Valide` ou `Ecart accepte` doit lier `OpenAPI` + `ContextEnvelope` + artefact `CREOS` reviewable, conformement au cadrage.
3. Tant que `Preuve / validation` est vide, la ligne ne peut pas passer a `Valide`.
4. Les formulations `A confirmer sur http://localhost:4445` doivent etre remplacees par des valeurs exactes une fois le slice verrouille.
